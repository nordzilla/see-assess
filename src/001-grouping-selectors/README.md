# Grouping Selectors

Selectors can be grouped to share the same rules by
using a comma to list them before declaring the rules.

---

### CSS
```CSS
h1, h2, h3 {
  color: red;
}

```

---

### HTML
```HTML
<!DOCTYPE html>
<html>
  <head>
    <title>Grouping Selectors</title>
    <link rel="stylesheet" href="style.css" />
  </head>
  <body>
    <h1>Hello</h1>
    <h2>There</h2>
    <h3>World!</h3>
  </body>
</html>

```
